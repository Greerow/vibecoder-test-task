# VibeCoder Test Task

## Возможности

- Анализ клиентских заявок
- Определение срочности
- Выделение контактов
- Обработка нескольких заявок в одном сообщении
- Защита от prompt injection
- Обработка нецелевых сообщений

## Запуск

1. Скопировать application.properties.example
2. Создать application.properties
3. Заполнить токены
4. Запустить run.bat
