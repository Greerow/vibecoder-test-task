# VibeCoder Test Task

## Возможности

- Анализ клиентских заявок
- Определение срочности
- Выделение контактов
- Обработка нескольких заявок в одном сообщении
- Защита от prompt injection
- Обработка нецелевых сообщений

## Запуск

1. Скопировать application.properties.example в application.properties
2. Заполнить токены
3. Запустить run.bat
